# 任务类型分类

## web 类型

**适用场景**: Web 应用开发、前端项目、全栈项目

**常见任务**:
- 搭建网站、博客、管理系统
- 前端页面开发
- API 接口开发

**拆解策略**:
1. 项目初始化
2. 框架配置
3. 页面开发
4. API 开发
5. 部署配置

## cli 类型

**适用场景**: 命令行工具、脚本、自动化工具

**常见任务**:
- 开发 CLI 工具
- 编写自动化脚本
- 系统管理工具

**拆解策略**:
1. 参数解析
2. 核心功能实现
3. 错误处理
4. 帮助文档
5. 打包发布

## api 类型

**适用场景**: API 服务开发、后端接口

**常见任务**:
- RESTful API
- GraphQL API
- 微服务接口

**拆解策略**:
1. 数据库设计
2. 接口定义
3. 业务逻辑
4. 认证授权
5. 测试文档

## tool 类型

**适用场景**: 开发工具、辅助脚本

**常见任务**:
- 代码生成器
- 数据转换工具
- 测试工具

**拆解策略**:
1. 需求分析
2. 工具设计
3. 核心功能
4. 测试验证
5. 文档编写

## config 类型

**适用场景**: 配置管理、环境设置

**常见任务**:
- 项目配置
- CI/CD 配置
- 环境变量管理

**拆解策略**:
1. 现状分析
2. 配置设计
3. 实现配置
4. 验证测试
5. 文档更新
